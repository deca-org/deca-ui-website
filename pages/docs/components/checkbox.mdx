# Checkbox
Checkbox component is used in forms when a user needs to select multiple values from several options.

## Import
```
import { Checkbox } from "../../../deca-ui/module/index"
```
## Usage
<ComponentWrapper>
    <Checkbox initialCheck label="Checkbox" />
</ComponentWrapper>
```
<Checkbox initialCheck label="Checkbox" />
```

## Disabled
Use the `disabled` prop to disable the Checkbox.
<ComponentWrapper>
  <Checkbox disabled label="Checkbox" />
  <Checkbox initialCheck disabled label="Checkbox" />
</ComponentWrapper>
```
<Checkbox disabled label="Checkbox" />
<Checkbox initialCheck disabled label="Checkbox" />
```

## Sizes
Use the `size` prop to change the size of the Checkbox. You can set the value to `sm`, `md`, or `lg`.
<ComponentWrapper>
  <Checkbox size="sm" initialCheck label="Small" />
  <Checkbox size="md" initialCheck label="Medium" />
  <Checkbox size="lg" initialCheck label="Large" />
</ComponentWrapper>

```
<Checkbox size="sm">Small</Checkbox>
<Checkbox size="md">Medium</Checkbox>
<Checkbox size="lg">Large</Checkbox>
```

## Colors
Use the `color` prop to change the color of the Checkbox. You can set the value to `primary`, `secondary`, `success`, `warning` or `error`.
<ComponentWrapper>
  <Checkbox initialCheck color="primary" label="Primary" />
  <Checkbox initialCheck color="secondary" label="Secondary" />
  <Checkbox initialCheck color="success" label="Success" />
  <Checkbox initialCheck color="warning" label="Warning" />
  <Checkbox initialCheck color="error" label="Error" />
</ComponentWrapper>

```
<Checkbox initialCheck color="primary" label="Primary" />
<Checkbox initialCheck color="secondary" label="Secondary" />
<Checkbox initialCheck color="warning" label="Warning" />
<Checkbox initialCheck color="success" label="Success" />
<Checkbox initialCheck color="error" label="Error" />
```
